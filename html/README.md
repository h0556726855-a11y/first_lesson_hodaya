סטודיו לעיצוב, מיתוג ואיור דיגיטלי


🎨 על הפרויקט:
אתר זה הוא תיק עבודות  המיועד להציג שירותי עיצוב גרפי, עימוד מקצועי, מיתוג עסקי ואיור דיגיטלי.
האתר נבנה כחלק מפרויקט לימודי ב-HTML ו-CSS, במטרה ליצור נוכחות דיגיטלית מרשימה המשלבת אסתטיקה אמנותית עם פונקציונליות חווייתית.


🛠 טכנולוגיות בשימוש
HTML5: מבנה סמנטי של דפי האתר (אודות, שירותים, צור קשר).

CSS3: עיצוב מתקדם הכולל:

Flexbox & Grid: לסידור אלמנטים ורספונסיביות.

CSS Animations: יצירת המסגרת הדינמית המשתנה.

Variables: לניהול פלטת הצבעים המשתנה של המותג.

Fonts: הטמעת פונטים נקיים ומודרניים לקריאות מקסימלית על רקע כהה.


💡 החלטות מרכזיות בפרויקט
1. בחרתי ברקע שחור מלא כבסיס לאתר כדי לאפשר לעבודות האיור והמיתוג "להאיר" מתוך המסך.
   הרקע הכהה מפחית עומס ויזואלי ומעניק תחושה יוקרתית וחדשנית.
2. מסגרת דינמית : במקום להשתמש בתמונות רקע כבדות, הטמעתי מסגרת חיה המבוססת על 3 צבעי המותג.
   המסגרת זזה בלופ אינסופי כדי לשדר יצירתיות, תנועה ודינמיות – ערכים המייצגים את עולם הפרסום והאיור.
3. מבנה היררכי: דף ה"אודות" תוכנן כך שהמידע יוגש במנות קטנות  כדי להקל על המשתמש לסרוק את תחומי המומחיות שלי במהירות.


🚀 קשיים והתמודדות
הקושי:
 צירת אנימציית המסגרת הזזה (Gradient Border) בצורה חלקה שתקיף את כל תוכן הדף בלי לשבור את המבנה שלו.
הפיתרון:
שימוש ב-pseudo-elements (כמו ::before) עם conic-gradient ואנימציית סיבוב (rotate), תוך שימוש ב-inset כדי ליצור את אפקט המסגרת הדקה והמדויקת שנראתה בסקיצה.
